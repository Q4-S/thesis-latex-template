A modern dissertation (or thesis) LaTeX template using pdfLatex or XeLaTeX (app.tex as main file), inspired by https://webgs.jnu.ac.kr/GradTreatise/Method. Fully customizable and clickable table of contents (ToC), list of figures (LoF), list of tables (LoT), appendices, supports one or two-sided printing, and Korean/Hangeul fonts. You could use it to write any books in English, Indonesian or Korean, especially if you're a student. You are most welcome to contribute or suggest any improvements at https://github.com/Q4-S/thesis-latex-template
