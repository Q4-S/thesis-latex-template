A modern dissertation (or thesis) LaTeX template using pdflatex or XeLaTeX (app.tex as the main file), inspired by https://webgs.jnu.ac.kr/GradTreatise/Method. Fully customizable and clickable table of contents (ToC), list of figures (LoF), list of tables (LoT), appendices, supports one or two-sided printing, and Korean/Hangeul fonts. You can extend its use to write any book in English, Indonesian, Korean or a mix of them. You are most welcome to contribute or suggest any improvements at https://github.com/Q4-S/thesis-latex-template
