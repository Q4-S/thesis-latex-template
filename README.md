# thesis-latex-template

A modern dissertation (or thesis) LaTeX template using XeLaTeX (NOT pddflatex, need to change default compiler first before running/compile the main file app.tex), inspired by https://webgs.jnu.ac.kr/GradTreatise/Method. Fully customizable and clickable table of contents (ToC), list of figures (LoF), list of tables (LoT), appendices, supports one or two-sided printing, and Korean/Hangeul fonts. You could use it to write any books in English, Indonesian or Korean, especially if you're a student. You are most welcome to contribute or suggest any improvements
